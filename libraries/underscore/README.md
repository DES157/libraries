# Underscore.js

[Underscore](http://underscorejs.org/) is a utility library.

Functions are categorized for use with:

* Collections
* Arrays
* Functions
* Objects
* Utility

Functions may be used in either an object-oriented or functional style [[ref](http://underscorejs.org/#oop)].
